Lunch Roulette PWA

How to publish with QR:
1. Go to Netlify Drop: https://app.netlify.com/drop
2. Drag this ZIP file into Netlify Drop.
3. Copy the website link Netlify gives you.
4. Create a QR code from that website link.
5. Put the QR on your poster, Teams post, pantry wall, or email blast.

Install on phone:
- iPhone: Open the site in Safari > Share > Add to Home Screen.
- Android: Open in Chrome > menu > Add to Home screen / Install app.

Important:
- A QR code needs a hosted website link.
- Netlify gives HTTPS automatically, which is needed for PWA install features.
- To update foods later, edit index.html and redeploy the ZIP.
